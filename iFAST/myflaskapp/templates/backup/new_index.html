iFAST(Issue Fast Automated Identification Tool)

iFAST is an application built with the intention of automating the process of executing combination of backend scripts to aid multiple teams.
Basically its built with three modules for Phase1 with below mentioned features:
  • Ⅰ ) Knowledge Base:
    • Module is intended for creating/contributing to Knowledge.
    • User can walk through all the issues. Can edit/update the issues if required.
  • Ⅱ ) Issue Look Up:
    • This Module Helps users to search particular issue/ set of issues.
    • User can then execute individually or can execute all the issues.
  • Ⅲ ) Configuration Management(Future Releases):
    • This Module Helps users to configure the different set of configuration to aid the executions.
  • Ⅳ ) User Management(Future Releases):
    • This Module Helps superadmin to manage the users of the tool.
